什么是HeyChain?

HealthbeautyChain，简称HeyChain,打造全球首个医疗美容全产业联盟链，利用区块链分布式储存技术对医美药品流通、医院资质、医师资质，培训机构资质、消费者数据等进行去中心化数据存储，整合金融保险机构，为消费者提供可靠、可信、便捷的医美消费生态服务平台。

项目技术

HEYChain联盟链底层架构将采用分布式账本技术框架，并兼容超级账本 Fabric 全球标准。Fabric是数字事件(交易)的账簿，在HeyChain上，这个账簿将由多个参与者共享，每个参与者都在系统中拥有权益。账簿只有在所有参与者达成共识的情况下才能够更新，并且信息一旦记录就永远不能修改。HeyChain将解决医美行业的药品生产流通、医师从业资格证注册、医疗机构资质注册、手术全程数据记录、出现问题溯源等问题

关于本项目

医美市场前景巨大：随着颜值经济崛起以及消费升级，医疗美容行业蓬勃发展，全球医疗美容行业平均年增长在10%以上，有分析师预测2018年总项目容量将达到约8,000万例。

虚拟现实

美籍华人Tina Mars 身兼世界华商青商联盟秘书长、世界华商纽约总会秘书长，是国际时尚传媒业界精英、华尔街区块链投资项目主要倡导人之一，公关能力范围辐射全球四十余个国家。Tina作为HEY项目联合创始人，将在美国曼哈顿建立全球运营中心，主要负责HEY项目的全球实施和推广。

电子商务

HEY目前已经建立起强大的社区，包括微信群、qq群、电报群、头条号、公众号等，粉丝数量达10万左右。建立起了100多人的推广团队，全球币圈社群覆盖超过100万人，其中中国大陆地区QQ群、微信群、电报群覆盖超过10000个。

区块链

HEY最大优势就是落地快，目前，项目已经与5家正规医疗美容整形医院正式签署战略合作协议，与上百家医美机构达成合作意向。HEY将具体应用到医美机构及医师资质查询、药物数据监管、整形预测系统、电子病历系统、第三方数据库等商业场景。

社交网络和博客圈

项目在硅谷拥有研发中心，全球运营总部位于曼哈顿，已有中国内地和美国洛杉矶等地技术团队超过30人，合作机构十几家。各国项目落地合伙人也在招募中。
